## Usage

### Run Project Locally

Assumes local installation of [Node.js](https://nodejs.org/en/download/)

To run the project locally:

- Clone or fork this repository.
- Run `npm install`
- Run `npm run react-dev` (If you made chenges to any of the files)
- Run `npm run start`
